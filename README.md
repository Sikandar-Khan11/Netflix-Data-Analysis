# Netflix Data Analysis

## Project Overview

This project analyzes the Netflix Movies and TV Shows dataset using Python. The goal is to explore the dataset, clean the data, visualize trends, and discover meaningful insights.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Dataset

The dataset contains information about Netflix movies and TV shows, including:

- Title
- Type (Movie or TV Show)
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Genre
- Description

## Project Steps

1. Loaded the dataset
2. Explored the dataset
3. Cleaned missing values
4. Checked for duplicate records
5. Performed exploratory data analysis (EDA)
6. Created visualizations
7. Generated insights

## Visualizations

- Movies vs TV Shows
- Top 10 Countries
- Ratings Distribution
- Release Year Distribution

## Key Insights

- Netflix contains more Movies than TV Shows.
- The United States contributes the highest number of titles.
- TV-MA and TV-14 are among the most common ratings.
- Most content in the dataset was released after 2015.
